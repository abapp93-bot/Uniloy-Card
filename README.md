# UNILOY Digital Business Card

Free GitHub Pages digital business card for Andrea Bruno.

## Publish on GitHub Pages
1. Create a public repository named `uniloy-card`.
2. Upload all files in this folder.
3. Go to Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: `main`, folder `/root`.
6. Open: `https://abapp93-bot.github.io/uniloy-card/`.
